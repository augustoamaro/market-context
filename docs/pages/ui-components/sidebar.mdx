# Sidebar

Arquivo: `app/components/Sidebar.tsx`.

## Funções principais

- Watchlist com favoritos em `localStorage` (`hsc_favorites`)
- Shortcut `⌘K` / `Ctrl+K` para busca de símbolo
- Atualização de ticker a cada 30s (`/api/ticker`)
- Scanner de melhor setup a cada 60s (`/api/context` em chunks)
- Estrela para setup com score >= 60
