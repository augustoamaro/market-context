- `lib/binance/client.ts`: fetch de klines + cache.
- `lib/services/marketContext.ts`: contexto completo por ativo/timeframe.
- `lib/decision.ts`: steps, score e sinal.
- `app/components/CurrentSignalCard.tsx`: sinal final + snapshot.
