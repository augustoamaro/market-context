# Requirements

## Runtime

- Node.js >= 20
- pnpm >= 9

## Dados de mercado

O dashboard funciona com endpoints públicos de OHLCV da Binance.
`BINANCE_API_KEY` é opcional e ajuda apenas com limites de requisição.

## Ambiente local

1. Clonar o repositório.
2. Rodar instalação de dependências.
3. Rodar o app principal em `localhost:3000`.
4. Rodar docs em `localhost:3001`.
