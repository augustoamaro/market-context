# Quick Start

## App principal

```bash
pnpm install
pnpm dev
```

Abrir `http://localhost:3000`.

## Docs (Nextra)

```bash
pnpm docs:dev
```

Abrir `http://localhost:3001`.

## Notas

- O app principal funciona sem `BINANCE_SECRET`.
- Para docs, a pasta `docs/` roda como pacote separado no workspace.
