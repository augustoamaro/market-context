# Add Timeframe

1. Inclua o timeframe em `TIMEFRAMES` (`lib/config.ts`).
2. Valide se o endpoint de klines da Binance suporta esse intervalo.
3. Confirme impacto no carregamento paralelo de `app/page.tsx`.
4. Ajuste layout/legibilidade do `TrendMonitorCard` se necessário.

Mais timeframes aumentam custo de requisição e carga de processamento.
